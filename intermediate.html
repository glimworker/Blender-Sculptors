Task 3: Alien Pet!

Your First Creature Design

🎯 Your Mission

Design a friendly alien pet that's never been seen before! It can have any number of eyes, legs, or tentacles — let your imagination run wild!

🔧 Tools You'll Learn

  • Combining multiple shapes
  • Dynamic Topology (Dyntopo) for adding details
  • Smoothing and texturing
  • Creating unique features

📝 Step by Step

  1. Create base body: Add > Mesh > UV Sphere for body. Enable Mirror Modifier for symmetry. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.25
  2. Shape main form: Use Grab tool to pull body into alien shape. Use Draw tool to create muscle forms. Smooth tool to blend shapes. Inflate to add volume where needed
  3. Add limbs/tentacles: Snake Hook to pull out limb shapes. Grab tool to position limbs. Crease to create joint areas. Clay Strips to build muscle definition
  4. Create facial features: Mask tool to mark eye areas. Snake Hook to pull out eye stalks (if desired). Draw tool to shape mouth/sensory organs. Smooth to refine all features
  5. Add alien details: Clay Strips to create skin texture. Draw tool to add patterns/markings. Crease to define important edges. Smooth for final polish

💡 Hot Tips!

  • Think about how your alien would move
  • Make it look friendly — this is a pet!
  • Try to make it different from Earth animals
  • Give it a personality through its features

🌟 Extra Challenges

  • Add unusual textures to its skin
  • Give it special features like glowing spots
  • Create a unique way for it to eat or move

✨ Show & Tell

When you're done, be ready to:

  • Tell us about your alien's special features
  • Explain how it would interact with its owner
  • Show your favourite detail you created

Task 5: Magic Cookie Jar!

Magical Container Design

🎯 Your Mission

Create a magical jar that could hold enchanted cookies! Add mysterious patterns, magical symbols, and cool decorations.

🔧 Tools You'll Learn

  • Creating circular shapes
  • Making repeating patterns
  • Adding surface details
  • Designing decorative elements

📝 Step by Step

  1. Create jar body: Add > Mesh > Cylinder. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.20. Use Grab tool to shape the jar's profile — wider belly, narrower neck
  2. Shape the lid: Add a separate UV Sphere. Flatten tool for the base. Draw tool to add a handle on top. Smooth to refine the shape
  3. Add magical patterns: Draw tool to create pattern outlines around the sides. Crease tool to define sharp pattern edges. Clay Strips to build up raised designs
  4. Create mysterious symbols: Draw tool for rune-like shapes. Crease for sharp detail. Smooth to blend edges where needed
  5. Surface interest: Clay Strips for texture. Scrape tool for aged/worn areas. Smooth for final polish

💡 Hot Tips!

  • Think about how the lid fits
  • Make patterns that go all the way around
  • Add details that make it look magical
  • Consider what kind of cookies it holds!

🌟 Extra Challenges

  • Add glowing runes or symbols
  • Create a special locking mechanism
  • Make it look ancient and mysterious

✨ Show & Tell

When you're done, be ready to:

  • Explain what makes your jar magical
  • Show your favourite pattern
  • Tell us what kind of cookies it holds

Task 6: Dinosaur Head!

Prehistoric Creature Design

🎯 Your Mission

Create an awesome dinosaur head! You can make it scary like a T-Rex, friendly like a Brachiosaurus, or invent your own species!

🔧 Tools You'll Learn

  • Shaping strong features
  • Creating scales and textures
  • Making teeth and horns
  • Adding character to eyes

📝 Step by Step

  1. Basic head shape: Add > Mesh > UV Sphere. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.20. Use Grab tool to elongate into a head shape
  2. Jaw and snout: Grab tool to pull out the snout. Crease to define the jaw line. Draw tool to build up the brow ridge
  3. Eye sockets and nostrils: Mask tool to mark eye areas. Draw tool (subtract mode) to push in eye sockets. Snake Hook for nostril openings. Smooth to refine edges
  4. Horns or crests: Snake Hook to pull out horn shapes. Grab to position and shape. Pinch tool to sharpen tips. Smooth to blend base into head
  5. Scales and texture: Clay Strips with small brush for scale patterns. Crease between scales for definition. Draw tool for smaller detail scales
  6. Teeth and details: Snake Hook to pull out tooth shapes. Pinch to sharpen points. Draw tool for gum line. Smooth for final polish

💡 Hot Tips!

  • Look at real dinosaur pictures for ideas
  • Think about what your dino eats — herbivores have flat teeth, carnivores have sharp ones
  • Make the scales different sizes — bigger on top, smaller near the mouth
  • Give it some personality in its expression

🌟 Extra Challenges

  • Add unusual features like feathers
  • Create a unique head crest
  • Mix features from different dinosaurs

✨ Show & Tell

When you're done, be ready to:

  • Tell us what kind of dinosaur you made
  • Show how you made the scales
  • Explain what makes your dino special

Task 8: Sea Creature!

Underwater Wonder

🎯 Your Mission

Design a creature that lives in the deepest parts of the ocean! It can be beautiful, strange, or a bit of both.

🔧 Tools You'll Learn

  • Creating flowing shapes
  • Making fins and tentacles
  • Adding sea creature textures

📝 Step by Step

  2. Add fins, tentacles, or arms: Snake Hook to pull out fin shapes or tentacles. Grab to position and curve them. Flatten tool to make fins thin and flat. Smooth to create flowing shapes
  3. Special features: Draw tool to create light organ bumps. Inflate for bulbous features like angler fish lures. Snake Hook for antennae or whiskers
  4. Face and eyes: Draw tool to build up eye area. Crease to define mouth line. Inflate for large deep-sea eyes. Smooth to refine expression
  5. Scales or patterns: Clay Strips for scale rows. Draw tool for stripe patterns. Crease for definition. Smooth for final polish

💡 Hot Tips!

  • Think about how it moves in water
  • Deep sea creatures often have special lights
  • Make shapes flow like they're underwater
  • Consider how it catches its food

🌟 Extra Challenges

  • Design unique hunting features

Task 9: Superhero Mask!

Design Your Hero Identity

🎯 Your Mission

Design an amazing superhero mask that shows off your hero's powers and personality!

🔧 Tools You'll Learn

  • Symmetrical design with Mirror Modifier
  • Creating dramatic shapes
  • Adding heroic details
  • Surface finishing techniques

📝 Step by Step

  1. Base mask shape: Add > Mesh > UV Sphere. Switch to Edit Mode [Tab]. Cut in half and delete the back. Add Mirror Modifier for symmetry. Switch to Sculpt Mode
  2. Shape the mask: Grab tool to shape around face contours. Draw tool to build up brow area. Smooth to create clean curves
  3. Eye openings: Mask tool [M] to draw eye shapes. Invert mask [Cmd+I on Mac]. Draw tool (subtract) to push in eye openings. Clear mask [Alt+M]. Crease to define eye edges sharply
  4. Heroic features: Snake Hook to pull out wing/horn shapes. Draw tool to create raised emblems. Crease for sharp heroic lines. Clay Strips to build up forehead crest
  5. Surface details: Draw tool for panel lines. Crease for sharp edges. Scrape for battle damage (optional). Smooth for final polish

💡 Hot Tips!

  • Think about what powers your hero has
  • The mask should match the hero's personality
  • Sharp lines = tech hero, flowing curves = nature hero
  • Add a symbol or emblem that represents the hero

🌟 Extra Challenges

  • Add retractable eye lenses
  • Create a built-in communication device
  • Design a mask that transforms

✨ Show & Tell

When you're done, be ready to:

  • Tell us about your hero and their powers
  • Show the special features of the mask
  • Explain why you chose this design

Task 10: Magic Tree!

Enchanted Nature

🎯 Your Mission

Create an enchanted tree that could exist in a magical forest! Think twisted trunks, mysterious hollows, and magical details.

🔧 Tools You'll Learn

  • Creating organic, twisting forms
  • Making realistic bark textures
  • Building branch structures
  • Adding magical elements

📝 Step by Step

  1. Create the trunk: Add > Mesh > Cylinder. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.20. Use Grab tool to twist and shape the trunk — make it gnarly and interesting
  2. Add branches: Snake Hook to pull out main branch shapes. Grab to position and curve branches. Pinch to taper branch ends. Smooth to blend branches into trunk
  3. Create bark texture: Clay Strips for bark ridges running vertically. Crease for deep bark cracks. Draw tool for smaller bark detail. Scrape for worn/smooth areas
  4. Hollow and roots: Draw tool (subtract) to create a mysterious hollow. Snake Hook to pull out root shapes at the base. Crease to define root edges
  5. Magical details: Draw tool to add glowing knots. Inflate for mysterious bumps or mushroom-like growths. Snake Hook for hanging vines. Smooth for final polish

💡 Hot Tips!

  • Real trees are never perfectly straight — add character!
  • Roots should spread out wide at the base
  • Think about what kind of magic lives in your tree
  • A hollow or doorway adds mystery

🌟 Extra Challenges

  • Add a face hidden in the bark
  • Create glowing fruits or flowers
  • Make it look like it could walk

✨ Show & Tell

When you're done, be ready to:

  • Tell us what kind of magic your tree has
  • Show how you made the bark look real
  • Explain what creature might live inside it

Task 12: Dragon Egg!

Mystical Object Design

🎯 Your Mission

Design a mysterious dragon egg! What kind of dragon is growing inside? The outside of the egg should give us clues!

🔧 Tools You'll Learn

  • Creating smooth egg forms
  • Adding scale-like surface patterns
  • Making cracks and emerging features
  • Surface detail techniques

📝 Step by Step

  1. Basic egg shape: Add > Mesh > UV Sphere. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.15 (finer detail for egg). Use Grab tool to elongate into an egg — slightly pointy at one end
  2. Scale pattern: Clay Strips with small brush to create overlapping scale shapes. Start from bottom, work up. Crease between scales for definition. Keep scales consistent in size
  3. Add cracks (optional): Draw tool (subtract) to carve crack lines. Crease to deepen cracks. Draw tool to add raised edges along cracks
  4. Emerging features: Snake Hook to pull out a small claw or snout from a crack. Draw tool to add glow-like raised areas. Smooth to blend naturally
  5. Final surface: Draw tool for veiny patterns. Inflate for warm/pulsing spots. Smooth to polish smooth areas between scales

💡 Hot Tips!

  • Think about what colour and type of dragon is inside
  • Scales should overlap like roof tiles
  • A few cracks make it more exciting
  • Maybe something is glowing from inside!

🌟 Extra Challenges

  • Add a baby dragon claw breaking through
  • Create veins of magical energy across the surface
  • Design a nest or stand for your egg

✨ Show & Tell

When you're done, be ready to:

  • Tell us what kind of dragon is inside
  • Show how you made the scale pattern
  • Explain how close it is to hatching

Task 14: Monster Cup!

Character Container Design

🎯 Your Mission

Create a cup that looks like a monster! It should be fun, a bit scary, and still look like you could actually drink from it.

🔧 Tools You'll Learn

  • Combining functional and character shapes
  • Creating facial features on objects
  • Handle design
  • Surface texture and detail

📝 Step by Step

  1. Basic cup shape: Add > Mesh > Cylinder. Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.20. Use Grab tool to shape the cup — wider at top, narrower base
  2. Monster face: Draw tool to build up brow ridges. Inflate for bulging eyes. Draw (subtract) for eye sockets. Snake Hook to pull out horns or ears
  3. Mouth and teeth: Draw tool to create lips around the rim. Snake Hook to pull out fang shapes from the rim. Pinch to sharpen tooth tips. Crease to define the mouth line
  4. Handle: Snake Hook to pull out a handle shape (or a tail!). Grab to curve it into a usable shape. Smooth to make it comfortable-looking
  5. Monster details: Clay Strips for bumpy monster skin. Draw tool for warts or spots. Crease for wrinkles. Smooth for final polish

💡 Hot Tips!

  • The cup still needs to work as a cup — keep the inside hollow!
  • Give your monster personality — friendly scary is best
  • The handle could be a tail, arm, or tentacle
  • Think about what it would feel like to hold

🌟 Extra Challenges

  • Add tiny monster arms hugging the cup
  • Create a matching saucer monster
  • Design a lid that looks like a monster hat

✨ Show & Tell

When you're done, be ready to:

  • Tell us what your monster drinks
  • Show the coolest feature
  • Explain how you'd hold it without getting scared!

Task 17: Magical Gem!

Crystal Formation Design

🎯 Your Mission

Craft a powerful magical crystal or gemstone! Think about what power it holds and how that shows on the outside.

🔧 Tools You'll Learn

  • Creating geometric crystal shapes
  • Making flat faceted surfaces
  • Adding magical energy details
  • Combining sharp and smooth forms

📝 Step by Step

  1. Basic crystal shape: Add > Mesh > Ico Sphere (or Cone for a pointed crystal). Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.15. Use Grab tool to elongate into a crystal point
  2. Facets and faces: Flatten tool to create flat crystal faces. Crease to define sharp edges between facets. Pinch to sharpen crystal ridges
  3. Crystal cluster (optional): Add additional smaller shapes around the base. Grab to angle them outward. Flatten and Crease for facets on each one
  4. Magical energy: Draw tool for energy veins running through the crystal. Snake Hook for wisps of magical energy. Inflate for glowing nodes at intersections
  5. Polish: Smooth for crystal face surfaces (keep edges sharp!). Crease for any final edge definition. Draw for tiny imperfections that make it look real

💡 Hot Tips!

  • Real crystals have flat faces and sharp edges
  • Think about what power your gem holds — fire? ice? time?
  • Imperfections make crystals look more real
  • A base or mount can make it look like a display piece

🌟 Extra Challenges

  • Create a crystal cluster with many different-sized crystals
  • Add an inclusion (something trapped inside)
  • Design a magical staff or wand with your gem mounted in it

✨ Show & Tell

When you're done, be ready to:

  • Tell us what power your gem holds
  • Show how you made the crystal faces
  • Explain where it was found

Task 20: Crazy Hat!

Wild Wearable Design

🎯 Your Mission

Design the wildest, most creative hat you can imagine! It could be for a wizard, an alien king, a deep-sea explorer, or anyone you can think of.

🔧 Tools You'll Learn

  • Creating wearable proportions
  • Building elaborate decorations
  • Combining multiple design elements
  • Advanced surface detailing

📝 Step by Step

  1. Base hat shape: Add > Mesh > Cylinder (for the crown) and a flattened Torus or Cylinder (for the brim). Switch to Sculpt Mode [Tab]. Enable Dyntopo: Detail Size 0.20. Use Grab to shape the crown — tall, short, pointy, droopy — go wild!
  2. Brim and structure: Grab tool to shape the brim — wavy, uneven, or dramatic. Flatten tool to keep brim surfaces clean. Smooth to blend brim into crown
  3. Decorations: Snake Hook to pull out feathers, tentacles, or antlers. Draw tool for buckles, bands, and trim. Clay Strips for fabric folds and draping. Crease for stitching lines
  4. Special features: Inflate for embedded gems or eyes. Draw tool for magical symbols. Snake Hook for dangling charms or chains
  5. Final details: Clay Strips for fabric texture. Scrape for worn leather effect. Crease for sharp detail. Smooth for final polish on smooth areas

💡 Hot Tips!

  • Think about who would wear this hat
  • Mix unexpected things — a top hat with tentacles? A crown with mushrooms?
  • It should still look like it could sit on a head
  • More details = more impressive — but don't forget to smooth!

🌟 Extra Challenges

  • Add a creature living in or on the hat
  • Create moving parts (like flaps or wings)
  • Design a hat that tells a story

✨ Show & Tell

When you're done, be ready to:

  • Tell us who would wear your hat
  • Show your most creative feature
  • Explain the story behind the design
