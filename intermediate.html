How to Use This Booklet

1. Pick what interests you - You don't have to do them in order

2. Complete at least 3-4 tasks - This builds your intermediate skills

3. It's okay to struggle - These are harder than beginner tasks

4. Go back if needed - Review beginner tasks if you get stuck

5. Experiment! - Try new techniques and make mistakes

Essential Keyboard Shortcuts for Intermediate Work

Sculpting:

  • D: Draw brush
  • G: Grab brush
  • M: Mask tool
  • Ctrl+I: Invert mask
  • F: Brush size
  • Shift+F: Brush strength

Advanced Tools:

  • E: Extrude
  • I: Inset
  • Ctrl+R: Loop cut
  • Shift+D: Duplicate
  • Ctrl+J: Join objects
