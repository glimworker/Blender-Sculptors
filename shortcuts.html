Blender Keyboard Shortcuts

Quick Reference Guide for All Tasks

Essential Navigation

Middle Mouse Rotate view
Shift + Middle Mouse Pan view
Mouse Wheel / Ctrl + Middle Mouse Zoom view
Numpad 1 Front view
Numpad 3 Side view
Numpad 7 Top view
Numpad 5 Toggle Perspective/Orthographic
Numpad . Focus on selected object

Mode Switching

Tab Toggle Edit Mode / Object Mode
Ctrl + Tab Mode pie menu
Z Shading mode menu
Shift + Z Toggle wireframe
Alt + Z Toggle X-Ray

Basic Tools

G Grab/Move
R Rotate
S Scale
X, Y, or Z Limit to axis (after G/R/S)
Shift + D Duplicate
Alt + D Linked duplicate
Ctrl + J Join objects
P Separate

Edit Mode Tools

E Extrude
I Inset
Ctrl + R Add loop cut
K Knife tool
F Create face/edge
Ctrl + B Bevel
Alt + Click Select edge loop
Ctrl + Click Select shortest path

Sculpt Mode Tools

D Draw brush
Shift + S Smooth brush
G Grab brush
C Clay brush
L Layer brush
M Mask brush
Ctrl + I Invert mask
Alt + M Clear mask

Selection Tools

A Select all
Alt + A Deselect all
B Box select
C Circle select
L Select linked
Shift + Click Add to selection
Ctrl + Click Remove from selection
1, 2, 3 Vertex, Edge, Face select mode

Sculpt Brush Settings

F Brush size
Shift + F Brush strength
Ctrl Invert brush effect
[ and ] Decrease/Increase brush size
Shift + [ and ] Decrease/Increase brush strength

File Operations

Ctrl + S Save file
Ctrl + Shift + S Save As
Ctrl + O Open file
Ctrl + N New file
Ctrl + Z Undo
Ctrl + Shift + Z Redo

Note: On Mac, replace Ctrl with Cmd (⌘) and Alt with Option (⌥) for most shortcuts.

If using a keyboard without a numpad, enable "Emulate Numpad" in Preferences > Input.
